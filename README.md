## 🚀 Projets Principaux

### 🌐 Application Web Full-Stack
> **Stack :** Python · Flask · PostgreSQL · PL/pgSQL

Développement d'un site web complet avec architecture MVC, API REST, gestion de sessions et base de données relationnelle. Mise en place d'une couche d'accès aux données avec `psycopg` et procédures stockées en PL/pgSQL.

---

### ⚙️ Compilateur & Assembleur
> **Stack :** C · Flex · Bison · NASM

Réalisation d'une **chaîne de compilation complète** : analyse lexicale (Flex), analyse syntaxique (Bison), arbre syntaxique abstrait, et génération de code assembleur NASM. Projet académique de haut niveau reproduisant les étapes fondamentales d'un vrai compilateur.

---

### 🗜️ Compresseur d'Images
> **Stack :** C · Algorithmique · Gestion mémoire

Implémentation d'un algorithme de compression avec optimisation des performances et gestion fine de la mémoire. Analyse de complexité temps/espace et comparaison de stratégies d'optimisation.

---

### 🎮 Jeu Vidéo 2D
> **Stack :** Java · POO · Design Patterns

Application des principes de la **Programmation Orientée Objet** : héritage, polymorphisme, interfaces, et design patterns (Factory, Observer). Développement du moteur de jeu et de la logique de gameplay.
